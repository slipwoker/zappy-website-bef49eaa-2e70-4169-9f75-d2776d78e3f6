How it works

Physics Molars empowers students and educators to learn by doing. Create custom physics scenes, set forces and constraints, then observe and explain the outcomes—a hands-on loop that transforms abstract concepts into visual, interactive understanding.

Create a scene

Start with a blank canvas or choose from templates. Add objects, surfaces, and structures using intuitive drag-and-drop tools—no coding required.

objects = [ball, ramp, spring]

Set properties

Define gravity, friction, mass, velocity, and other physical parameters. Adjust constraints like fixed points, springs, or collision behaviors to match real-world scenarios.

g = 9.81 m/s²

Explain what happened

Run the simulation and observe motion, forces, and energy transfers in real time. Use built-in analysis tools to measure outcomes and explain the physics principles at work.

Maps to what you already teach

Physics Molars aligns seamlessly with typical middle and high school physics sequences. Educators can integrate our interactive physics simulation tool into demos, labs, and formative assessments—building scenes that illustrate core concepts with consistent vector conventions and SI units (N,J,m/s).

Forces

Demonstrate Newton's laws with free-body diagrams and dynamic force vectors that respond to user input.

Energy

Visualize kinetic and potential energy transformations in custom-built scenes with real-time calculations.

Waves

Create wave interference patterns and explore frequency, wavelength, and amplitude interactively.

Electricity

Build circuit simulations showing current flow, voltage drops, and component behavior in parallel and series.

Momentum

Model collisions and conservation of momentum with adjustable mass and velocity parameters.

Vectors

Practice vector addition, decomposition, and component analysis with draggable arrows and coordinate grids.

All scenes use consistent vector conventions: arrows for forces, standard SI units (N,kg·m/s), and right-handed coordinate systems. This ensures students build transferable mental models across topics.
